## 🔐 ВАЖЛИВО ПРО TOKEN:

GitHub більше не приймає паролі для Git операцій.

**Створіть Personal Access Token:**
1. GitHub → Settings → Developer settings
2. Personal access tokens → Generate new token
3. Виберіть: repo (всі галочки)
4. Generate token
5. СКОПІЮЙТЕ TOKEN
6. Використовуйте його замість пароля
