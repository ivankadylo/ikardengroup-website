## 📦 ДЕ ЩО ЗНАХОДИТЬСЯ:

### **Додати товар:**
```
admin/admin.html → Кнопка ➕ → Заповнити → Зберегти
```

### **Завантажити фото:**
```
admin/admin.html → Вибрати товар → Галерея → Клік/Drag
```

### **Встановити знижку:**
```
admin/admin.html → Вибрати товар → Поле "Sleva %" → Зберегти
```

### **Переглянути замовлення:**
```
F12 → Application → Local Storage → ikarden-orders
```

### **Зберегти в Git:**
```
git-manager.bat → [1] Зберегти
```

### **Відновити версію:**
```
git-manager.bat → [3] Відновити
```
