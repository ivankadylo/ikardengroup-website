1Створіть GitHub акаунт

2Створіть репозиторій

  1. На GitHub натисніть "+" вгорі справа
  2. Виберіть New repository
  3. Заповніть форму:

Description: Сайт виробника меблів IKarden
☑ Public (публічний)
☐ НЕ додавайте README
☐ НЕ додавайте .gitignore
  1. Натисніть Create repository
  2. СКОПІЮЙТЕ URL який з'явиться:

3Запустіть автоматичний деплой

Знайдіть файл DEPLOY-NOW.bat в папці проекту і двічі клікніть на нього.

⚠️ Важливо!
Коли скрипт попросить URL - вставте той що скопіювали в Кроці 2.

4Увімкніть GitHub Pages

  1. На GitHub перейдіть у ваш репозиторій
  2. Натисніть Settings (вгорі)
  3. Ліворуч знайдіть Pages
  4. В розділі Source:
    • Branch: main
    • Folder: / (root)
  5. Натисніть Save
✅ Готово!
Через 1-2 хвилини сайт буде доступний на:
https://USERNAME.github.io/ikarden-website/

  4. Generate new token (classic)
  5. Налаштування:
