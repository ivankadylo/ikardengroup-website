## 💡 ПРИКЛАДИ:

### **Після додавання нових фото:**
```
1. Відкрити git-manager.bat
2. Вибрати [1] Зберегти
3. Написати: "Додано фото для товару Стіл"
```

### **Після великих змін:**
```
1. Відкрити git-manager.bat
2. Вибрати [1] Зберегти
3. Написати: "Змінено дизайн кошика"
```

### **Створити мітку версії:**
```
1. Відкрити git-manager.bat
2. Вибрати [5] Створити мітку
3. Написати: v1.1.0
4. Опис: Додано пошук товарів
```

### **Щось зламалось, треба повернутись:**
```
1. Відкрити git-manager.bat
2. Вибрати [2] Історія
3. Знайти останню робочу версію (наприклад: a1b2c3d)
4. Вибрати [3] Відновити
5. Вставити: a1b2c3d
```
