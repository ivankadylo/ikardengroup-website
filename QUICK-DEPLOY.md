## ⚡ ЗА 5 ХВИЛИН!

### 1️⃣ Створіть репозиторій на GitHub
```
https://github.com/new
Назва: ikarden-website
Public
Create repository
```

### 2️⃣ Відкрийте термінал і виконайте:
```bash
cd C:\Users\ivank\muscle-diary\ikarden-website

git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ВАШ_USERNAME/ikarden-website.git
git branch -M main
git push -u origin main
```

### 3️⃣ Увімкніть GitHub Pages
```
GitHub → Settings → Pages
Branch: main
Folder: / (root)
Save
```

### 4️⃣ Готово! 🎉
```
Ваш сайт: https://ВАШ_USERNAME.github.io/ikarden-website/
Через 1-2 хвилини буде онлайн!
```
