## 🚀 Швидкий Старт

### Локально

1. **Клонуйте репозиторій:**
```bash
git clone https://github.com/ВАШ_USERNAME/ikarden-website.git
cd ikarden-website
```

2. **Відкрийте сайт:**
```bash
# Просто відкрийте index.html у браузері
start index.html
```

### Деплой на GitHub Pages

1. **Запустіть автоматичний скрипт:**
```bash
DEPLOY-NOW.bat
```

2. **Або вручну:**
```bash
git add .
git commit -m "Initial deployment"
git push origin main
```

3. **Увімкніть GitHub Pages:**
   - Settings → Pages
   - Source: main branch
   - Save

🎉 **Готово!** Сайт буде доступний на: `https://USERNAME.github.io/ikarden-website/`

## 📝 Як Редагувати

### Варіант 1: Через Інтерфейс Сайту

1. Натисніть кнопку **🔧 Admin** (вгорі справа)
2. Увійдіть в режим адміністратора
3. Використовуйте кнопки **Upravit** для редагування
4. Зміни зберігаються автоматично

### Варіант 2: Через Код

1. **Змінити текст:**
   - Відкрийте `index.html`
   - Знайдіть потрібний текст
   - Змініть його
   - Збережіть файл

2. **Змінити стилі:**
   - Відкрийте `css/style.css`
   - Знайдіть `:root` для зміни кольорів
   - Змініть значення CSS змінних
   - Збережіть файл

3. **Оновити сайт:**
```bash
git add .
git commit -m "Оновлення контенту"
git push
```
