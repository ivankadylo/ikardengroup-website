# IKarden Website

Сайт виробника меблів IKarden.

🌐 https://ikardengroup.com

## Структура:
- `index.html` — головна сторінка
- `admin-panel.html` — адмін панель (об'єднана)
- `checkout.html` — оформлення замовлення
- `css/style.css` — стилі
- `js/script.js` — головний скрипт
- `js/products-data.js` — дані товарів
- `images/` — фотографії

## Оновлення сайту:
1. Зміни файли
2. Відкрий Git Bash
3. `git add . && git commit -m "update" && git push`
