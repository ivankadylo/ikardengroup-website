## 🆘 ТИПОВІ ПРОБЛЕМИ І РІШЕННЯ

### Проблема: "Press any key" без тексту
**Причина:** Українські букви в .bat файлі  
**Рішення:** Використовувати файли англійською (SIMPLE-DEPLOY.bat)

### Проблема: Git не встановлено
**Причина:** Git відсутній на системі  
**Рішення:** Завантажити з git-scm.com, встановити, перезавантажити

### Проблема: Permission denied при push
**Причина:** Потрібен Personal Access Token  
**Рішення:** Створити token на GitHub, використати як пароль

### Проблема: Repository not found
**Причина:** Репозиторій не створено на GitHub  
**Рішення:** Спочатку створити на github.com/new

### Проблема: Сайт показує 404
**Причина:** GitHub Pages не увімкнено  
**Рішення:** Settings → Pages → Branch: main → Save
