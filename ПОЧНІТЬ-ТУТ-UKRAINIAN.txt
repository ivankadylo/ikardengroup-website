  ЩО РОБИТИ ЗАРАЗ:
═══════════════════════════════════════════════════════════

КРОК 1: Перевірте систему
────────────────────────────────────────────────────────────
   Двічі клікніть на: CHECK-SYSTEM.bat
   
   Він перевірить:
   - Чи встановлено Git?
   - Чи ви в правильній папці?
   - Чи все готово?

КРОК 2: Завантажте на GitHub
────────────────────────────────────────────────────────────
   Двічі клікніть на: SIMPLE-DEPLOY.bat
   
   Він зробить:
   - Додасть файли до Git
   - Підключить GitHub
   - Завантажить ваш код

КРОК 3: Увімкніть GitHub Pages
────────────────────────────────────────────────────────────
   1. Відкрийте: https://github.com/ivankadylo/ikarden-website
   
   2. Натисніть: Settings (вгорі)
   
   3. Натисніть: Pages (зліва)
   
   4. Налаштуйте:
      Source: Deploy from a branch
      Branch: main
      Folder: / (root)
      Натисніть: Save
   
   5. Зачекайте 2 хвилини
   
   6. Ваш сайт: https://ivankadylo.github.io/ikarden-website/

2. Заповніть:

   Expiration: No expiration

3. Натисніть: Generate token

4. СКОПІЮЙТЕ ТОКЕН (більше не покажеться!)

5. Використайте токен як пароль
