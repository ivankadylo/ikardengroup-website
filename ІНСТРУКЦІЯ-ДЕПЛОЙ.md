## ⚡ ШВИДКИЙ МЕТОД (5 ХВИЛИН)

### ✅ ВСЕ ВЖЕ ГОТОВО! ПРОСТО ВИКОНАЙТЕ:

### **КРОК 1: Створіть GitHub акаунт** (якщо ще немає)

1. Відкрийте: https://github.com
2. Натисніть **Sign up**
3. Використайте email: **ikardengroup@gmail.com**
4. Створіть пароль
5. Підтвердіть email

### **КРОК 2: Створіть репозиторій**

1. На GitHub натисніть **"+"** вгорі справа
2. **New repository**
3. Заповніть:
   ```
   Repository name: ikarden-website
   Description: Сайт виробника меблів IKarden
   ☑ Public (щоб був доступний безкоштовно)
   ☐ НЕ додавайте README
   ☐ НЕ додавайте .gitignore
   ```
4. **Create repository**
5. **СКОПІЮЙТЕ** URL що з'явиться, наприклад:
   ```
   https://github.com/USERNAME/ikarden-website.git
   ```

### **КРОК 3: Запустіть автоматичний скрипт**

1. Знайдіть файл **DEPLOY-NOW.bat** в папці проекту
2. **Двічі клікніть** на нього
3. Коли попросить URL - **вставте** той що скопіювали
4. Коли попросить опис змін - натисніть **Enter**
5. Якщо попросить логін/пароль GitHub:
   - **Логін:** ваше ім'я на GitHub
   - **Пароль:** використайте Personal Access Token (див. нижче)

### **КРОК 4: Увімкніть GitHub Pages**

1. На GitHub перейдіть у ваш репозиторій
2. Натисніть **Settings** (вгорі)
3. Ліворуч знайдіть **Pages**
4. В розділі **Source:**
   - **Branch:** виберіть `main`
   - **Folder:** виберіть `/ (root)`
5. Натисніть **Save**

### **КРОК 5: Зачекайте 1-2 хвилини**

Сторінка оновиться і покаже:
```
✅ Your site is published at:
https://USERNAME.github.io/ikarden-website/
```

### **🎉 ГОТОВО! САЙТ ОНЛАЙН!**

## 🔐 ЯК ОТРИМАТИ PERSONAL ACCESS TOKEN

Якщо Git просить пароль, використовуйте Token:

1. GitHub → **Settings** (ваш профіль справа вгорі)
2. Прокрутіть вниз → **Developer settings**
3. **Personal access tokens** → **Tokens (classic)**
4. **Generate new token** → **Generate new token (classic)**
5. Заповніть:
   ```
   Note: ikarden-website-deploy
   Expiration: No expiration (або 90 days)
   ☑ repo (всі галочки в цій секції)
   ```
6. **Generate token**
7. **СКОПІЮЙТЕ TOKEN** (він більше не покажеться!)
8. Використовуйте його як пароль при push

### **Спосіб 2: Через VS Code онлайн**

1. На GitHub натисніть клавішу **"."** (крапка)
   АБО змініть URL: `github.com` → `github.dev`
2. Відкриється VS Code онлайн!
3. Редагуйте файли
4. Ліворуч натисніть іконку **Source Control**
5. Напишіть опис змін
6. **Commit & Push**
7. ✅ Сайт оновиться!

### **Спосіб 3: Локально**

1. Змініть файли на комп'ютері
2. Запустіть **DEPLOY-NOW.bat**
3. ✅ Готово!

## 🆘 ВИРІШЕННЯ ПРОБЛЕМ

### **Помилка: "fatal: not a git repository"**
```bash
cd C:\Users\ivank\muscle-diary\ikarden-website
git init
git add .
git commit -m "Initial commit"
```

### **Помилка: "Permission denied"**
- Перевірте що ввели правильний Token
- Створіть новий Token з правами `repo`

### **Сайт не оновлюється**
- Очистіть кеш браузера: `Ctrl + Shift + R`
- Зачекайте 5 хвилин
- Перевірте GitHub Actions (вкладка Actions)

### **Сайт показує 404**
- Перевірте Settings → Pages
- Переконайтесь що файл називається `index.html`
- Переконайтесь що branch = `main`
