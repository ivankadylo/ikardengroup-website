## 🐛 ТИПОВІ ПРОБЛЕМИ:

### Товари не показуються:
```
→ F5
→ Перевірити js/script.js завантажився
→ F12 → Console
```

### Зміни не зберігаються:
```
→ Перевірити localStorage (F12 → Application)
→ Спробувати інший браузер
```

### Фото не завантажуються:
```
→ Перевірити розмір (<5MB)
→ Перевірити формат (JPG, PNG)
```

### Слайдер не працює:
```
→ Переконатись що > 1 фото
→ F5
→ Очистити кеш
```
