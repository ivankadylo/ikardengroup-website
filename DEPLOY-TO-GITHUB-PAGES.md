## 📦 КРОК 2: ЗАВАНТАЖЕННЯ КОДУ НА GITHUB

### 2.1. Відкрийте термінал (Command Prompt або Git Bash)
```

Enter
```

### 2.2. Перейдіть у папку проекту:
```bash
cd C:\Users\ivank\muscle-diary\ikarden-website
```

### 2.3. Виконайте команди:

```bash
# 1. Ініціалізація Git (якщо ще не зроблено)
git init

# 2. Додати всі файли
git add .

# 3. Створити комміт
git commit -m "Initial commit - IKarden website ready"

# 4. Додати GitHub як remote
# ЗАМІНІТЬ "ВАШ_USERNAME" на ваше ім'я на GitHub!
git remote add origin https://github.com/ВАШ_USERNAME/ikarden-website.git

# 5. Перейменувати гілку на main (якщо потрібно)
git branch -M main

# 6. Завантажити на GitHub
git push -u origin main
```

### 2.4. Введіть облікові дані GitHub
```
Username: ваше_ім'я
Password: ваш_токен (або пароль)
```

**УВАГА:** Якщо просить пароль - використовуйте Personal Access Token:
1. GitHub → Settings → Developer settings
2. Personal access tokens → Tokens (classic)
3. Generate new token
4. Виберіть: repo (всі галочки)
5. Generate token
6. Скопіюйте токен і використайте як пароль

## 🆘 ВИРІШЕННЯ ПРОБЛЕМ

### Сайт не оновлюється:
```
1. Перевірте GitHub Actions (вкладка Actions)
2. Очистіть кеш браузера (Ctrl + Shift + R)
3. Зачекайте 5 хвилин
```

### Помилка при push:
```
1. Перевірте інтернет
2. Перевірте токен доступу
3. Спробуйте: git pull origin main
4. Потім: git push
```

### Сайт показує 404:
```
1. Перевірте що файл називається index.html
2. Перевірте Settings → Pages налаштування
3. Перевірте що branch = main
```
