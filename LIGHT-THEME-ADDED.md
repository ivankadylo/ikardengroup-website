## 🔧 **НАЛАШТУВАННЯ:**

### **Змінити кольори:**
```css
/* В css/style.css → :root */

/* Темна тема: */
:root {
    --accent-primary: #4ade80;  ← Змінити тут
}

/* Світла тема: */
body.light-theme {
    --accent-primary: #22c55e;  ← Змінити тут
}
```

### **Додати третю тему:**
```css
body.blue-theme {
    --bg-primary: #1e3a8a;
    --text-primary: #ffffff;
    --accent-primary: #60a5fa;
}
```

```javascript
function toggleTheme() {
    // Додати логіку для третьої теми
    if (body.classList.contains('light-theme')) {
        body.classList.remove('light-theme');
        body.classList.add('blue-theme');
    } else if (body.classList.contains('blue-theme')) {
        body.classList.remove('blue-theme');
    } else {
        body.classList.add('light-theme');
    }
}
```

## 🐛 **МОЖЛИВІ ПРОБЛЕМИ:**

### **Якщо тема не перемикається:**
```
1. F12 → Console
2. Перевірити помилки
3. Перевірити що toggleTheme() викликається
4. Перевірити що клас додається до body
```

### **Якщо не зберігається:**
```
1. Перевірити localStorage в DevTools
2. Application → Local Storage
3. Має бути: ikarden-theme = 'light' або 'dark'
```

### **Якщо кольори не міняються:**
```
1. Перевірити що використовуєте var(--змінна)
2. Не жорстко прописані кольори (#ffffff)
3. Hard refresh: Ctrl + Shift + R
```
