## 🆘 ПРОБЛЕМИ?

### **Помилка: Permission denied**
→ Створіть Personal Access Token (див. вище)

### **Помилка: Repository not found**
→ Перевірте що репозиторій називається `ikarden-website`

### **Сайт показує 404**
→ Перевірте Settings → Pages → Branch = main

### **Сайт не оновлюється**
→ Ctrl + Shift + R (очистити кеш)
→ Зачекайте 5 хвилин
→ Перевірте GitHub Actions
